Manejo de información espacial
con bases de datos SQL

SQL es el lenguaje más poderoso para hacer consultas de información. Con el paso de los años, estas consultas se han trasladado al plano de las bases de datos de información geográfica, basándose en complementos como PostGIS.

En este curso aprenderás desde los fundamentos del lenguaje SQL hasta variedad de ejemplos prácticos de aplicación para dar respuesta a diversas consultas geográficas, de forma que luego puedas aplicar dichos conocimientos constantemente en tu vida profesional.
